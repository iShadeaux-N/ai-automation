## Upload Checklist (Every Video)

- [ ] Script finalized (generated via `youtube_pipeline.py` or written manually)
- [ ] Voiceover recorded in ElevenLabs (Adam voice, 0.9x speed, settings in `elevenlabs_settings.txt`)
- [ ] Video assembled in CapCut/DaVinci using scene breakdown
- [ ] Thumbnail created (dark bg, high-contrast focal point, ≤3-word overlay)
- [ ] Title: curiosity gap + keyword, ≤70 chars
- [ ] Description: hook + summary + timestamps + affiliate links + AI disclosure
- [ ] Tags: 15 tags from SEO metadata file
- [ ] End screen: subscribe CTA + "Watch next" card
- [ ] AI disclosure in description: *"Narration in this video is AI-generated using ElevenLabs."*
- [ ] Video logged in `pivot_tracker.py` after 48 hours
